# 急診手術審查

這是一個可在瀏覽器中執行的單頁工具，用於匯入急診手術審查 Excel，標註不符合審查規範的項目，並提供審查、申訴與回覆欄位。

## 使用方式

1. 開啟 GitHub Pages 網頁。
2. 點選「Excel 資料檔」並選擇急診審查作業 `.xlsx`。
3. 檢視系統標註的不符合或待確認項目。
4. 填寫審查結果、審查備註、主治醫師申訴與申訴回覆。
5. 點選「匯出審查 CSV」保存審查結果。
6. 若要保存回顧紀錄，可先部署留言板後端，輸入後端網址與讀取密碼後點選「上傳審查資料」。

本工具只在瀏覽器端讀取原始 Excel。按下「上傳審查資料」時，只會把已載入分析後的審查資料轉成 JSON 傳給你設定的後端。

## 留言板後端

可使用 `google-apps-script-backend.js` 建立簡易 Google 試算表後端：

1. 建立一份 Google 試算表。
2. 開啟 Apps Script，貼上 `google-apps-script-backend.js` 的內容。
3. 部署為 Web App，執行身分選「我」，存取權限選可使用此網站的人。
4. 將 Web App URL 填入主頁的「留言板後端網址」與回顧頁同名欄位。
